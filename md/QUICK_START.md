# 🚀 快速启动指南

## ✅ 依赖已修复，应用已就绪！

所有缺失的文档处理包都已安装并记录到 `requirements.txt` 中。

---

## 启动方式（三选一）

### 🎯 方式 1️⃣：推荐 - 智能启动脚本
```powershell
.\start_simple.ps1
```
✨ **优点**：自动检查环境，显示访问地址，错误提示清晰

---

### 🎯 方式 2️⃣：调试启动（如有错误显示详细信息）
```batch
start_debug.bat
```
✨ **优点**：任何错误都会显示，窗口不会自动关闭

---

### 🎯 方式 3️⃣：手动启动
```bash
# 激活虚拟环境
.\venv\Scripts\Activate.ps1

# 启动Flask应用
python run.py
```
✨ **优点**：完全控制启动过程

## 📤 文档上传功能位置

在**管理员仪表板**的左侧卡片中找到：
- **"管理快捷操作"** 卡片
- 点击 **"📤 文档上传"** 按钮
- 弹出上传窗口

### 支持的文件类型
✅ Excel (.xlsx, .xls)  
✅ PDF (.pdf)  
✅ Word (.docx)  
✅ 文本文件 (.txt)

## 🔧 如果仍有问题

1. **应用还是不能启动？**
   - 使用 `start_debug.bat` 查看详细错误信息
   - 参考 `STARTUP_FIX_GUIDE.md` 文档

2. **找不到 "📤 文档上传" 按钮？**
   - 确保已以 admin 账号登录
   - 确保在管理员仪表板页面（/admin/dashboard）

3. **上传文件后无法识别数据？**
   - 确保文件中包含必要的字段（如：地区、人口数量、机构名称等）
   - 查看浏览器控制台或Flask日志中的错误信息
